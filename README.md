 
> **Udacity · Power BI Nanodegree · Project 2** 
 
A boardroom-ready Power BI report comparing Lapdog and Lapcat device performance for the fictional pet startup, Waggle.
 
---
 
## Overview
 
For this project, we were tasked with building a report that tells the story of how Lapcat data compares to findings from the dog collar Lapdog devices. Both products are owned by Waggle, a fictional pet startup.
 
---
 
## CEO Questions
 
The report directly addresses two questions from the CEO:
 
1. Did average daily steps increase for cats wearing the Lapcat device, as they did for dogs with Lapdog?
2. Were Lapcat device owners as satisfied with their product as Lapdog owners?
 
---
 
## Stakeholder Requests
 
### Chief Marketing Officer
The report must be "on-brand" — using only the Waggle color palette, logo, and approved company icons.
 
### Product Team
Demographic comparisons between dogs and cats, family ownership insights, and slicers for self-serve exploration.
 
---
 
## Project Requirements
 
- [x] Identify fields to design metrics addressing the CEO's step and satisfaction questions
- [x] Visualize average daily steps over time and device satisfaction ratings (Lapcat vs. Lapdog)
- [x] Demographic insights using: bar chart, line chart, donut chart, table/matrix, scatter plot, bubble map, and card
- [x] Three-page layout: CEO questions → dog/cat differences → family ownership data
- [x] Branding elements incorporated per CMO request
- [x] Five slicers per page — drop-down, slider, hierarchy, Select All, and search-enabled variants
- [x] Two bookmarks: one to swap visuals dynamically, one to reset all filters
- [x] Navigation buttons with hover color/size response
